## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** v18 or later and **npm**
- **Python 3** and **pip**
- **Tesseract OCR** — required for document scanning features

<Warning>
  Tesseract OCR must be installed separately. Download and install it from the
  [official GitHub releases page](https://github.com/tesseract-ocr/tesseract).
  The ML Flask server will fail to start without it.
</Warning>

---

## Setup

<Steps>

  <Step title="Clone the repository">
    Clone the project and navigate into the `WEB SITE` folder, which contains both the frontend and the backend.

    <CodeGroup>

    ```bash Terminal
    git clone https://github.com/RJPOLICE_HACK_1202_CyberCrew_9/repo.git
    cd "WEB SITE"
    ```

    </CodeGroup>
  </Step>

  <Step title="Install frontend dependencies">
    From the root of the `WEB SITE` folder, install the React/Vite frontend dependencies.

    <CodeGroup>

    ```bash Terminal
    npm i
    ```

    </CodeGroup>

    This installs packages including React 18, React Router DOM, Framer Motion, styled-components, and Vite.
  </Step>

  <Step title="Install backend dependencies">
    Navigate into the `BACKEND` folder and install the Express backend dependencies.

    <CodeGroup>

    ```bash Terminal
    cd BACKEND
    npm i
    ```

    </CodeGroup>

    This installs Express, Mongoose, bcrypt, jsonwebtoken, cors, cookie-parser, body-parser, and nodemon.
  </Step>

  <Step title="Install Python ML dependencies">
    Install the Python packages required by the Flask ML server. Run this from anywhere with Python 3 available.

    <CodeGroup>

    ```bash Terminal
    pip install pillow joblib flask scikit-learn xgboost pandas PyMuPDF
    ```

    </CodeGroup>

    These packages power the phishing email classification and fraud legitimacy scoring models.
  </Step>

  <Step title="Install Tesseract OCR">
    <Warning>
      This step is required. The ML server uses Tesseract to extract text from uploaded documents and images.
    </Warning>

    Download and run the installer for your operating system from the
    [Tesseract OCR GitHub releases page](https://github.com/tesseract-ocr/tesseract).

    After installation, verify Tesseract is on your system `PATH`:

    <CodeGroup>

    ```bash macOS / Linux
    tesseract --version
    ```

    ```powershell Windows
    tesseract --version
    ```

    </CodeGroup>
  </Step>

  <Step title="Start all three services">
    The platform requires three processes running concurrently. Open three separate terminal windows.

    <Note>
      The three services run on separate ports:
      - **Frontend (Vite):** `http://localhost:5173`
      - **Express backend:** `http://localhost:8383`
      - **Flask ML server:** `http://localhost:5000`
    </Note>

    **Terminal 1 — Frontend (Vite dev server)**

    Run from the root of the `WEB SITE` folder:

    <CodeGroup>

    ```bash Terminal 1
    npm run dev
    ```

    </CodeGroup>

    **Terminal 2 — Express backend**

    <CodeGroup>

    ```bash Terminal 2
    cd BACKEND
    npm run dev
    ```

    </CodeGroup>

    This starts the Express server on port 8383 using nodemon, which watches for file changes and restarts automatically.

    **Terminal 3 — Flask ML server**

    <CodeGroup>

    ```bash Terminal 3
    cd "ML Models"
    python flaskserver.py
    ```

    </CodeGroup>

    This starts the Flask server on port 5000, which serves the fraud detection and phishing classification models.
  </Step>

  <Step title="Access the app and log in">
    Open your browser and navigate to:

    ```
    http://localhost:5173
    ```

    You will be redirected to the **Cyber Crime Portal** login page at `/login`.

    - **New users:** Click **Sign Up** to register an account. Select your role during registration (Victim, Police, Bank, or Court).
    - **Returning users:** Enter your credentials and click **Sign In**.

    After authentication, you are redirected to `/dashboard`, where the UI adapts to your assigned role.
  </Step>

</Steps>
